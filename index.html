Petualangan Sholat Hebat

Game Edukasi Interaktif Fiqih Sholat (Safinatun Najah) untuk Anak SMP/MTs.

🎮 Casual RPG Puzzle 📱 Android Offline APK 👦 Usia 12-15 Tahun

Misi Utama Game

Bagian ini menjelaskan esensi dari "Petualangan Sholat Hebat". Aplikasi ini dirancang tidak hanya untuk menguji, tetapi juga untuk mengajarkan materi fiqih sholat dengan pendekatan yang relevan bagi Gen-Z. Interaksi di sini memberikan gambaran cepat tentang flow pengguna.

📖

Materi Otentik

Berdasarkan kitab Safinatun Najah meliputi wudhu, tayamum, syarat, rukun, hingga sujud sahwi.

🕹️

Interaksi Beragam

Mini games seperti drag & drop, puzzle gerakan, dan simulasi troubleshooting kesalahan sholat.

🏆

Sistem Evaluasi

Reward koin, energi petir, dan quotes motivasi untuk membangun semangat belajar.

Identitas Visual

Visual game sangat menentukan ketertarikan anak SMP. Bagian ini merangkum arahan seni (art direction) yang menggabungkan elemen islami elegan dengan UI modern. Klik pada kartu warna untuk menyalin kode HEX-nya.

Aset & Spesifikasi Teknis

Informasi teknis dan library prompt AI yang siap di-copy-paste untuk mempercepat produksi game menggunakan AI Builder atau tim art/sound.

Spesifikasi Engine

  • ✔ Engine: Unity 2D / Construct 3 (Optimal kompresi APK).
  • ✔ Database: Local JSON / SQLite. Tanpa server.
  • ✔ Optimasi: Audio `.ogg`, Sprite Sheet Atlas, Asset 720p/1080p scaled.
  • ✔ Target Size: < 80 MB.
  • ✔ Keamanan: Enkripsi Base64 sederhana untuk save data.
